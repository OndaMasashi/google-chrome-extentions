# NotebookLM Toolkit

NotebookLM のノートブック一覧を強化する Chrome 拡張機能です。

## 機能

### 作成日順ソート
NotebookLM 標準の「新しい順」は**更新日順**ですが、この拡張で**作成日順**に並び替えできます。

- 「新しい順」の横に **「作成日順 ↓」** ボタンが追加されます
- クリックで降順/昇順をトグル

### タイトル検索（フィルタ）
ノートブックをタイトルでリアルタイム絞り込みできます。

- 検索フィールドにテキストを入力すると、マッチしないカードが非表示になります
- 空にすると全カードが復元されます
- ソートと組み合わせて使えます

## インストール方法

1. このリポジトリをクローンまたはダウンロード
2. Chrome で `chrome://extensions` を開く
3. 右上の **デベロッパーモード** を ON にする
4. **「パッケージ化されていない拡張機能を読み込む」** をクリック
5. `nlm-toolkit` フォルダを選択

## 技術仕様

- Chrome Manifest V3
- Content Script のみ（バックグラウンドスクリプト・ポップアップなし）
- 特別な権限不要（`notebooklm.google.com` でのみ動作）

### DOM変更への耐性

Google による DOM 構造変更に強い設計を採用しています:

- CSS クラス名に**一切依存しない**
- 日付テキスト (`YYYY/MM/DD`) のパターンマッチで要素を検出
- ノートブック URL (`/notebook/UUID`) の構造で要素を特定
- コンテナは「最も多くのカードを直接子要素に持つ祖先」として動的に検出
