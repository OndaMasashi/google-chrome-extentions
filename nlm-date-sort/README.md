# NotebookLM Sort by Creation Date

NotebookLM のノートブック一覧に「作成日順」ソート機能を追加する Chrome 拡張機能です。

NotebookLM 標準の「新しい順」は**更新日順**ですが、この拡張を使うと**作成日順**で並び替えできます。

## インストール方法

1. このリポジトリをクローンまたはダウンロード
2. Chrome で `chrome://extensions` を開く
3. 右上の **デベロッパーモード** を ON にする
4. **「パッケージ化されていない拡張機能を読み込む」** をクリック
5. `nlm-date-sort` フォルダを選択

## 使い方

1. [NotebookLM](https://notebooklm.google.com/) を開く
2. ノートブック一覧ページの右上、「新しい順」の横に **「作成日順 ↓」** ボタンが表示される
3. クリックすると作成日の降順（新しい順）でソート
4. もう一度クリックすると昇順（古い順）に切り替わる

## 技術仕様

- Chrome Manifest V3
- Content Script のみ（バックグラウンドスクリプト・ポップアップなし）
- 特別な権限不要（`notebooklm.google.com` でのみ動作）

### DOM変更への耐性

Google による DOM 構造変更に強い設計を採用しています:

- CSS クラス名に**一切依存しない**
- 日付テキスト (`YYYY/MM/DD`) のパターンマッチで要素を検出
- ノートブック URL (`/notebook/UUID`) の構造で要素を特定
- コンテナは「最も多くのカードを直接子要素に持つ祖先」として動的に検出
